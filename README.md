# 🎟️ 現場售票顯示與控制系統

這是一個用於現場活動的即時售票顯示系統，由兩個主要頁面組成： - **控制端
(`control.html`)**：用於售出與退回票券，具備確認彈窗防止誤操作。 -
**顯示端
(`display.html`)**：即時顯示各區剩餘票數，所有票售完時會自動顯示「票已全部售完」。

系統以 **Node.js + Socket.IO**
為基礎，可在局域網（LAN）運行，適合校園或現場活動使用。

------------------------------------------------------------------------

## 🧩 系統結構

    ticket_system/
    ├── server.js # Node.js 主伺服器（Express + Socket.IO）
    ├── public/ # 前端靜態資源資料夾
    │ ├── control.html # 控制端：售票、退票與即時統計
    │ ├── display.html # 顯示端：公開票數顯示
    │ ├── style.css # 全站統一風格（奶油風 / 溫柔光影風格）
    │ └── banner.png # 顯示端上方橫幅圖像
    └── package.json # npm 套件設定

------------------------------------------------------------------------

## 🚀 使用說明

### 1️⃣ 安裝環境

確保已安裝 **Node.js (v18+)**

``` bash
npm install express socket.io
```

------------------------------------------------------------------------

### 2️⃣ 啟動伺服器

``` bash
node server.js
```

啟動後，終端機會顯示：

    ✅ Ticket System running at http://localhost:5000

------------------------------------------------------------------------

### 3️⃣ 頁面說明

#### 🎛️ 控制端 (`control.html`)

-   每個票區顯示：
    -   總數
    -   已售出數量
    -   剩餘票數
-   有兩個操作按鈕：
    -   **售出一張** → 送出前會跳出確認框\
    -   **退回一張** → 送出前會跳出確認框\
-   修改後即時同步至顯示端。

------------------------------------------------------------------------

#### 🎟️ 顯示端 (`display.html`)

-   即時顯示各區剩餘票數。

-   若所有票區剩餘 0 張，會自動顯示：

        🚫 票已全部售完

-   數字變動時會有放大閃爍動畫。

-   票數為 0 的區域文字自動變紅。

------------------------------------------------------------------------

### 4️⃣ 同網段裝置同步

確保控制端與顯示端的裝置都在同一個區網（Wi-Fi 或有線網）。 -
控制端輸入： `http://你的電腦IP:5000/control.html` - 顯示端輸入：
`http://你的電腦IP:5000/display.html`

-   若要公開展示，建議搭配 **全螢幕模式（F11）**。
